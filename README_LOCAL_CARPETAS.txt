USO LOCAL - CARPETAS AUTOMATICAS PRIZE

1. Ejecuta app.py.
2. En el panel Administrador entra a: Subir documentos.
3. Presiona: Crear carpetas + detectar.
4. Se creará automáticamente la carpeta:
   DOCUMENTOS_PRIZE_AUTO
   al costado de app.py.

Ejemplo:
Si app.py está en:
D:\BOLETAS_PRIZE\app.py
la carpeta será:
D:\BOLETAS_PRIZE\DOCUMENTOS_PRIZE_AUTO\

Coloca los PDFs dentro de sus subcarpetas:
- DOCUMENTOS DE PAGO
- DOCUMENTOS DE LA EMPRESA
- DOCUMENTOS PERSONALES

Para trabajadores, el archivo o ruta debe contener el DNI de 8 dígitos.
El sistema solo cargará documentos de trabajadores activos.

Luego presiona: Sincronizar carpeta.
