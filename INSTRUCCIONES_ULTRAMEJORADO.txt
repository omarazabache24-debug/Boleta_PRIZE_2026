VERSIÓN ULTRAMEJORADA - GESTIÓN DOCUMENTAL / VACACIONAL / CONTRATOS

1) CORRECCIÓN PYTHON-DOCX
- Se agregó python-docx==1.1.2 en requirements.txt.
- En Render: Manual Deploy -> Clear build cache & deploy.
- En local: pip install -r requirements.txt

2) PLANTILLAS WORD
- Use campos en Word con comillas angulares: «NombreCompletoTrabajador», «Dni», «Cargo», «Planilla», etc.
- El sistema también mantiene compatibilidad con {{CampoOrigen}}.
- Al cargar un .docx, el sistema detecta automáticamente los campos y los registra en Campos.

3) CONDICIONES
- Las condiciones se evalúan contra datos reales del trabajador.
- El valor ahora permite desplegables para campos conocidos y escritura manual para otros campos.
- Para fechas y números, el campo cambia a tipo fecha/número.

4) FIRMA / RECONOCIMIENTO FACIAL / FIRMA DIGITAL
- Se habilitó panel: Gestión Contratación -> Firma / Facial / Digital.
- Se agregaron tablas firma_configuracion y firma_solicitudes.
- La conexión real RENIEC o firma digital requiere proveedor autorizado, API oficial y credenciales válidas.

5) PLANTILLA MAESTRA
- Se incluye PLANTILLA_MAESTRA_GESTION_RRHH_PRIZE.docx para usar como base de:
  Gestión documental, gestión vacacional, gestión contratos, firma y trazabilidad.

6) SOBRE ADAPTA / SANTA AZUL
- El enlace público redirige a autenticación, por lo que no se pudo extraer información interna sin credenciales.
- Se dejó el flujo preparado en base a las pantallas y requerimientos compartidos.
