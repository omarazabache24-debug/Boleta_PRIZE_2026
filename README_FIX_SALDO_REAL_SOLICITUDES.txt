MEJORA APLICADA - SALDO REAL EN SOLICITUDES

1. La solicitud ya no permite registrar vacaciones si el periodo seleccionado no tiene saldo real disponible.
2. El saldo real descuenta solicitudes pendientes/aprobadas para evitar usar el mismo saldo dos veces.
3. Se agregó control interno de periodos seleccionados por solicitud.
4. En la pantalla del trabajador ahora se muestra Saldo real y Usado/Pendiente por periodo.
5. Se mantiene el bloqueo de fecha inicio menor a la fecha actual.
6. Los registros y respaldos Excel locales siguen actualizándose automáticamente.
