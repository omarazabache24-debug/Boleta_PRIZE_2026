OPTIMIZACIÓN APLICADA

1. Se retiraron PDFs de ejemplo y __pycache__ del ZIP para reducir peso.
2. La carpeta DOCUMENTOS_PRIZE_AUTO queda creada con estructura liviana y se puede regenerar desde el sistema.
3. La cámara de Firma / Facial / Digital NO se activa al cargar la página; solo inicia al hacer clic en Activar cámara.
4. La captura de cámara fue bajada a 640x480 para consumir menos memoria en celular/laptop y Render Free.
5. Procfile optimizado para Render Free: 1 worker, 2 threads, timeout 120.
6. Se agregaron .renderignore y .gitignore para no desplegar cache, PDFs locales, uploads ni ZIPs.

IMPORTANTE EN RENDER:
- Subir este ZIP limpio o actualizar GitHub con estos archivos.
- Manual Deploy > Clear build cache & deploy.
- Si usa cámara desde celular, debe abrir el enlace HTTPS de Render.
