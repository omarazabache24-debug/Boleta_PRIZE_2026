MEJORA IMPLEMENTADA - APROBACIONES POR JEFE INMEDIATO

1. Se agregó una opción visible para usuarios: Gestión Vacacional > Aprobaciones jefe.
2. Todo trabajador cuyo DNI figure en la columna JEFE INMEDIATO de la plantilla de saldos podrá ver solicitudes asignadas para aprobar.
3. La solicitud ahora exige que el periodo seleccionado tenga DNI de jefe inmediato configurado; si no lo tiene, no registra y muestra aviso.
4. El panel de aprobaciones muestra: pendientes, aprobadas/GTH y rechazadas.
5. En el admin de vacaciones se agregó la columna DNI jefe en la lista de solicitudes para validar rápidamente a quién fue asignada.

IMPORTANTE:
- En la plantilla de saldos, JEFE INMEDIATO debe contener el DNI del jefe, no solo el nombre.
- El jefe debe existir como trabajador/usuario para poder iniciar sesión y aprobar.
