VERSIÓN ULTRAMEJORADA - PLANTILLAS POR GESTIÓN + FIRMA

CAMBIOS REALIZADOS:
1. Se mantienen la plantilla general de TRABAJADORES.
2. Se agregan plantillas separadas por cada gestión:
   - PLANTILLA_GESTION_DOCUMENTAL.xlsx
   - PLANTILLA_GESTION_VACACIONAL.xlsx
   - PLANTILLA_GESTION_CONTRATACION.xlsx
3. Se agregan rutas de descarga dentro del sistema:
   - /admin/plantilla_gestion/documental
   - /admin/plantilla_gestion/vacacional
   - /admin/plantilla_gestion/contratacion
4. En el menú se agregan accesos para descargar plantillas por módulo.
5. En Contratación se agrega sección/pestaña: Firma / Facial / Digital.
6. La pestaña de firma permite:
   - Visualizar flujo sugerido para contratos.
   - Enviar documentos de contratación a firma.
   - Registrar método: reconocimiento facial, firma digital, facial + firma digital, OTP o carga manual RR.HH.
   - Guardar trazabilidad en tabla firma_solicitudes.
7. Se conserva el panel de configuración para proveedor/API de firma:
   - /admin/firma/configuracion

IMPORTANTE SOBRE RENIEC / FIRMA DIGITAL:
El sistema queda preparado funcionalmente para integrarse con reconocimiento facial, RENIEC o firma digital. Para una validación real se requiere contratar/integrar un proveedor autorizado o API oficial, con credenciales y términos legales correspondientes.

EJECUCIÓN LOCAL:
1. Instalar dependencias: pip install -r requirements.txt
2. Ejecutar: python app.py
3. Usuario admin demo: admin / admin123

RENDER:
Mantener variables recomendadas:
- SECRET_KEY
- PERSIST_DIR=/data
